# pavelsvarc.com

Personal portfolio and project showcase website.

## About

Minimalistic landing page with dynamic project tiles. Each tile links to a separate project — built and maintained as fun side projects.

## Tech Stack

- HTML / CSS / JavaScript (frontend)
- PHP (backend API)
- PostgreSQL (database)
- Hosted on webglobe.cz

## Local Development

1. Clone the repo
2. Serve `public/` with any local server (e.g. `php -S localhost:8000 -t public`)
3. Copy `.env.example` to `.env` and fill in your database credentials

## Deployment

Pushes to `main` are automatically deployed to webglobe.cz via GitHub Actions + SFTP.
